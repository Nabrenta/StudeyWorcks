Постановка задачі

Реалізувати зв*язки між компонентами системи.

Програмна реалізація

Розробити для деякіх класів системи конструктори за замовченням, ініціалізації, копії та перетворення типів, тобто:


Для класу Library:
Конструктор за замовченням; конструктор ініціалізації; конструктор копії.


Для класу Catalog:
Конструктор ініціалізації; конструктор копії.


Для класу UserList:
Конструктор ініціалізації; конструктор копії.


Для класу Librariar:
Конструктор за замовченням; конструктор перетворення типів, конструктор копії.


Для класу ReadingRoom:
Конструктор за замовченням; конструктор копії.


Для класу Book:
Конструктор за замовченням; конструктор копії.


Для класу User:
Конструктор за замовченням; конструктор копії.


Для класу PrintScanner:
Конструктор за замовченням; конструктор копії.


Також для всіх класів необхідно розробити деструктори.
