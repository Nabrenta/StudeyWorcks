Автор:
Абрашина Надія, Група ІС-52, 2 курс, 2016 рік, Тема: "Моделювання роботи науково-технічної бібліотеки ВНЗ"

Проекти знаходяться у папці project (проекти мовою с++ знаходяться у папці с++, проекти мовою c# знаходяться у папці c#).
Виконувані файли знаходяться у папці run (окремо для с++ та для c#), а також у папках проектів.
Звіти для кожної версії знаходяться в папці report, в окремих папках, назва кожної з яких відповідає номеру версії (versionX, де X - номер версії) разом з діаграмами для відповідної версії.
Кожна сторінка відповідає одному пункьу звіту для відпоідної версії. Сторінки мають назви pNM.htm, де N - номер пункту, M - номер версії.
Технічне завдання (файл TZ.pdf) разом з головною сторінкою (файл index.htm) знаходяться у папці Report.
